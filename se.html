WHAT IS SOFTWARE ENGINEERING ?

Software Engineering is the discipline of building software in a way that is reliable, maintainable, and produced by a team. The course covered the software development lifecycle, requirements engineering, architecture, and academic writing for software topics, alongside a full project where engineering practices were applied end-to-end.

Reflection

Software Engineering was less about code and more about how teams actually build software. The academic writing assignment taught me how to research and write formally — a skill I underestimated at the time but have used many times since. The group project showed me how requirements engineering, architecture decisions, and team coordination determine whether a project succeeds.

Looking back, this course was the bridge between writing programs and engineering systems. It is the reason I now think of software as a team activity, not a solo one.
