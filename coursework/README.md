# Coursework

Source files for projects and assignments showcased in the eportfolio.

Organised by semester and subject. Each project lives in its own folder so the
folder URL can be linked directly from the eportfolio site.

```
coursework/
├── sem-5/        Semester 5 — AI, Database Programming, ESDM, PCS 2
└── sem-6/        Semester 6 — App Dev, BI, HPDP, Entrepreneurship, Data Engineering
```

Each project folder contains the report PDFs, code, screenshots, and a short
README describing what is inside.
