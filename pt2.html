WHAT IS PROGRAMMING TECHNIQUE 2 ?

Programming Technique 2 builds directly on first-year programming — introducing more advanced topics like object-oriented thinking, structured problem solving, and real project work. The course focused on putting techniques together in a single full project deliverable that combined design, implementation, and presentation.

Reflection

PT2 built directly on first-year programming but moved much closer to real software — bigger programs, more design thinking, and the first time I had to plan code before writing it. The project pushed me to combine multiple techniques into one deliverable, which was harder than any individual assignment in PT1.

Looking back, this was where I started thinking about code in terms of design and structure, not just syntax. The discipline of planning before coding stuck with me, and I still use the habit in every project today.
